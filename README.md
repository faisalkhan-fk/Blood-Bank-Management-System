# 🩸 Blood Bank Management System

A web-based Blood Bank Management System developed using PHP and MySQL. This project helps manage blood donors, blood requests, and administrator operations efficiently.

## 🚀 Live Demo

🔗 https://blood-bank-management-system-production-b781.up.railway.app

## ✨ Features

- 👤 Donor Registration
- 🩸 Blood Donation Management
- 📋 Donor List
- 📨 Blood Request System
- 🔐 Admin Login
- 🗑️ Add, Update & Delete Donor Records
- 📱 Responsive Design
- 🖼️ Image Upload Support

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Railway (Deployment)
- Git & GitHub

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/faisalkhan-fk/Blood-Bank-Management-System.git
```

2. Create a MySQL database.

3. Import the SQL file.

4. Configure the database credentials in `blood/connection.php`.

5. Run the project using XAMPP or deploy it on Railway.
